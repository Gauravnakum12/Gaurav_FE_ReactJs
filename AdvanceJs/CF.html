Web-Browser and Web-server

Web-Browser:-

  • Web-Browser use to view web pages, interact with websites, and access various online content.
  • Ex. Google Chrome, Mozilla Firefox, Microsoft Edge,Safari

    • Web-server:-

  • A web server is foundation for hosting and delivering websites and web applications over the internet.
  • . It receives requests from clients and responds by delivering the requested web pages or resources.

    • URL:-Universal resource locater

    Absolute Path:-

  • An absolute URL is a complete web address that includes all the necessary information to locate and access a specific resource on the internet.
  • Absolute URLs provide the full path from the protocol to the resource.

    • Relative Path:-

  • A relative URL is a way of specifying the location of a resource on the web in relation to the current document or the current base URL
  • A relative URL is a resource starting from the root domain.

    • Web site

  • It is collection of web-pages.

    • Static:-

  • A static website is a type of website that delivers content exactly as it is stored to the server .
  • static website are pre-built and remain unchanged after after visited by clint's.
  • Ex. Wikipedia

    • Dynamic:-

  • dynamic websites use server-side technologies to create and deliver content dynamically based on various factors, such as user input, database queries, or external data sources.

    • Home page and Webpage

    Home page:-

  • First page of every website it called Home page .
  • The home page typically provides an overview of the website's content, navigation options, and may include key information or features.

    • Web page:-

  • Web page is created using html css and javascript

    • W3C

  • W3c means World Wide Web Consortium .

  • Its mission is to lead the web to its full potential by promoting the development of high-quality standards and guidelines.
